# Masterarbeit_vogel


Model_training folder: 

FO-731_Training_JSN-prediction_1.ipynb: training notebook to train CNN with image and numeric data

FO-731_Training_JSN-prediction_2.ipynb: training notebook to train CNN with image and 4 most important numeric features

FO-731_Training_JSN-prediction_3.ipynb: training notebook to train CNN only on image data

JSN_XGBoost_machine.ipynb: training XGBoost model with numeric data


Pre-processing folder:

Class_def_20JSN_and_KL-grade.ipynb: Calculating JSN, and class, label data (20% JSN and KL-grad)

FO-739_XML_Extraction.ipynb: Calculating JSN, and class, label data (10% JSN)

Create_Masterfiles.ipynb: Compilation of datasets, which were used for training

Pre-process_JSN_images.ipynb: Pre-processing of the image data (crop, pre-processing) and normalization of the numeric data
